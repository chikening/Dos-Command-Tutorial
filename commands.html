Command Description Attributes
DIR Lists the files and directories in the current directory. /W (wide list format), /P (pause after each screen of information), /S (lists files in specified directory and all subdirectories).
CD (CHDIR) Changes the current directory. .. (moves up one directory level), <directory name> (navigates to a specific directory).
MD (MKDIR) Creates a new directory. No specific attributes.
RD (RMDIR) Removes a directory. /S (removes all directories and files in the specified directory in addition to the directory itself), /Q (quiet mode, does not ask for confirmation).
COPY Copies one or more files to another location. /Y (suppresses prompting to confirm you want to overwrite an existing destination file), /V (verifies that new files are written correctly).
XCOPY Copies files and directory trees. /E (copies directories and subdirectories, including empty ones), /I (if in doubt always assume the destination is a folder), /H (copies hidden and system files also).
DEL (ERASE) Deletes one or more files. /P (prompts for confirmation before deleting each file), /F (forces deletion of read-only files).
TYPE Displays the contents of a text file. No specific attributes.
EDIT Starts the MS-DOS Editor, a text editor used to create and edit ASCII text files. No specific attributes. (Not available in 64-bit versions of Windows).
MOVE Moves one or more files from one directory to another directory. No specific attributes.
REN (RENAME) Renames a file or files. No specific attributes.
ATTRIB Changes or views the attributes of one or more files. +R (sets the read-only file attribute), -R (clears the read-only file attribute), +H (sets the hidden file attribute), etc.
FORMAT Formats a disk for use with DOS. /Q (quick format), /FS (specifies the file system).
CHKDSK Checks a disk and displays a status report. /F (fixes errors on the disk), /R (locates bad sectors and recovers readable information).
DISKCOPY Copies the contents of one floppy disk to another. No specific attributes.
SYS Copies DOS system files and command interpreter to a disk. No specific attributes.
FIND Searches for a text string in a file or files. /I (ignores the case of characters when searching), /N (displays line numbers with the displayed lines).
MORE Displays one screen of output at a time. Used in conjunction with another command, e.g., DIR | MORE.
TREE Graphically displays the directory structure of a drive or path. /F (displays the names of the files in each directory), /A (uses ASCII characters).
XCOPY32 Same as XCOPY, used in older versions of DOS/Windows. Attributes are similar to XCOPY.
LABEL Creates, changes, or deletes the volume label of a disk. No specific attributes.
VOL Displays a disk volume label and serial number. No specific attributes.
PATH Displays or sets a search path for executable files. No specific attributes.
EXIT Quits the CMD.EXE program (command interpreter) or the current batch script. No specific attributes.
PAUSE Suspends processing of a batch file and displays a message. Often used to prompt the user for input.
DATE Displays or sets date No specific Attribrutes
Help Displays all Commands Available No specific Attribrutes
